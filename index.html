My Work

Middle-Where

Middle-Where is a fullstack application that allows multiple users to share their locations to find a spot in the middle as a good meet up spot. It uses GoogleMap's API to display an interactive maps to allow users to see all locations. It is currently functional, but many features are planned to be added in the near future to improve UX.
